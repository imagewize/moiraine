## Content Writing Guidelines

### Voice and Tone:
- **Confident:** Moiraine is the best choice for performance-focused WordPress users
- **Specific:** Use numbers (89+ patterns, 100% PageSpeed, 0.8s load time)
- **Action-oriented:** Every section should drive toward download or exploration
- **Technical but accessible:** Mention performance metrics but explain benefits

### Key Messages to Reinforce:

1. **Pattern Library Size:** 89+ patterns (3x more than competitors)
2. **Performance:** 100% PageSpeed scores, sub-1-second load times
3. **No-Code Editing:** Full Site Editor integration, visual customization
4. **Professional Quality:** Designed by experts, tested for accessibility
5. **Free and Open Source:** No lock-in, no subscription fees

### Keywords to Include:
- WordPress Site Editor
- Full Site Editing (FSE)
- Block theme
- Pattern library
- Performance optimization
- Core Web Vitals
- PageSpeed score
- Accessibility (WCAG AAA)
- No code / Visual editor
- 89+ patterns
